## Zyfun 源格式要点
- 标准 CMS JSON: `{code, msg, page, pagecount, limit, total, list[]}`
- 电影条目: `{vod_id, vod_name, type_id=1, type_name="电影", vod_en, vod_time, vod_remarks, vod_class, vod_actor, vod_content, vod_pic}`
- 电视剧条目: `{vod_id, vod_name, type_id=2, type_name="电视剧", vod_remarks 含季数和集数}`
- vod_pic: `https://image.tmdb.org/t/p/w500{poster_path}`
- vod_remarks: `⭐{评分} | {类型} | {演员}`

## 关键发现
- 付费视频平台（优酷/腾讯/Netflix）无法直接获取播放地址（DRM+反爬）
- 但 TMDB 提供完整的元数据（标题/简介/评分/封面/演员/类型）
- 用 TMDB 元数据在已有 CMS 资源站中搜索匹配，间接实现"浏览付费平台内容，免费平台播放"
- 67 个 CMS site 中 56 个支持搜索（`search=true`），搜索格式 `?ac=detail&wd=关键词`
- HTTP → HTTPS 自动升级（Zyfun 要求）
- 所有 URL 必须经过 curl 测试（10 workers, 8s 超时），只保留 200 的
- DRPY 类型（type=7, api=csp_XXX）无法 curl 测试，保留不测
- channel 直播流来自同一推流服务器则全部失效需清空
