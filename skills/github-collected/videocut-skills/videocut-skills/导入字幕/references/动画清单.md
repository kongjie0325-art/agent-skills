# 文字动画常用清单（已校验真实可用）

名称来自 `pyJianYingDraft/metadata/text_intro.py` 等，必须精确匹配。

## 文字入场（--anim，全部免费）

**通用爆款：**
- **冲屏位移**
- **放大**
- 弹入
- 渐显
- 开幕
- 卡拉OK

**滑入类：**
- 向上滑动 / 向下滑动 / 向左滑动 / 向右滑动
- 向上飞入 / 向下飞入

**擦除类：**
- 向上擦除 / 向下擦除 / 向左擦除 / 向右擦除
- 圆形扫描

**打字机类：**
- 复古打字机
- 居中打字
- 故障打字机
- 打字机 I / 打字机 II / 打字机 III / 打字机IV

**特殊：**
- 弹弓 / 弹簧 / 弹性伸缩
- 翻动 / 水平翻转 / 向上翻转
- 模糊 / 扭曲模糊
- 日出 / 溶解 / 闪动 / 生长 / 收拢 / 甩出 / 水墨晕开

## 文字出场（--out-anim）

- 淡出 / 缩小 / 向上滑出 / 向下滑出
- 复古打字机出场
