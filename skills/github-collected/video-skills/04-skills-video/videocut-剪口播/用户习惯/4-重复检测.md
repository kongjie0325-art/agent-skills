# 重复检测（句间 / 句内 / 纠正）

> 🚨 **核心：说了两遍，删前面，保后面。适用于所有粒度。**

---

## 一、句间重复

相邻句子（被静音≥0.5s分隔）开头相同≥5字，删**前句整句**。

```javascript
if (curr.text.slice(0, 5) === next.text.slice(0, 5)) {
  markAsError(curr);  // 删前句，保留后句
}
```

**隔一句重复**：中间是残句（≤5字）时，也要比对前后句。

| 前句 | 后句 | 删除 |
|------|------|------|
| "这是我剪出来的一个案例" | "这是我剪出来的一个案例" | 前句 |
| "好我们接下来开始怎么去" | "好我们接下来开始怎么去做一个剪口拨" | 前句 |

**多次重复**（≥3次）：保留最后完整的，前面全删。

---

## 二、句内重复

同一句内，短语 A 出现两次（中间夹杂 1-3 字）。**只删前面的片段，不删整句。**

```
A + 中间字 + A
↑只删这段↑    ↑保留↑
```

```
"于是很把于是很容易把它理解成一种不够友好但很高效的界面"
 ↑删这4字↑ ↑保留后面完整内容↑

❌ 整句标记删除
✓ 只删"于是很把"，保留"于是很容易把它理解成..."
```

**不是口误的情况**：列举（任务1任务2任务3）、强调（一个一个地）

---

## 三、重说纠正

说错后立即纠正，删前面错误的部分。

| 类型 | 原文 | 删除 |
|------|------|------|
| 部分重复 | 你再关你关掉 | "你再关" |
| 否定纠正 | 它是它不是 | "它是" |
| 词被打断 | 依赖[静]依赖关系 | "依赖[静]" |

---

## 易错点

```
❌ 删后句（更完整的）→ 违反"删前保后"
❌ 句内重复时删整句 → 应只删重复片段
❌ 逐字扫描找重复 → 应先分句再比对
❌ 只比对相邻句 → 也要比对隔一句（中间可能有残句）
```
