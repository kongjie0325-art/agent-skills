# 错词字典（用户自填）

> 没有口播稿时，用这份字典告诉 Agent 哪些专名容易被识别错。
> 在 Step 2「修正」环节，Agent 会对照这份字典逐条替换。

## 格式

左边是**常见误识别**，右边是**正确写法**，用 `→` 连接。

```
误识别 → 正确
```

## 品牌名/产品名（示例）

```

```

## 怎么用

1. 复制这份文件到你自己的项目目录（或直接在这里改）
2. 把你常踩的专名错误写进去
3. 跑导入字幕时，告诉 Agent "字典在 `xxx/错词字典.md`"
4. Agent 校对时会逐条对照

## 和口播稿的区别

| | 口播稿 | 错词字典 |
|---|---|---|
| 准确率 | 99% | 95%+ |
| 原理 | 字幕文本直接用稿子 | 识别后按字典替换 |
| 适用 | 照稿念的内容 | 即兴/半即兴 |
| 维护成本 | 每条视频一份 | 一份字典复用所有视频 |

两个可以一起用——有口播稿走口播稿，字典作为兜底。
