# AI Agent 发展时间线（2024 → 2026）

## 2024 年：萌芽期

| 时间 | 事件 | 影响 |
|------|------|------|
| 2024 Q2 | **Anthropic 发布 Claude Code（第一版）** | 最早的终端 AI 编码 Agent 之一 |
| 2024 Q3 | **Anthropic 提出 MCP（Model Context Protocol）** | 定义了 Agent ✕ 工具的标准化协议 |
| 2024 Q3 | **Cline（原 Claude Dev）开源** | VS Code 扩展 + Human-in-the-Loop 模式 |
| 2024 Q4 | **Aider 成为最流行 CLI Agent** | 41K Stars，Git-native 工作流 |
| 2024 Q4 | **第一个社区 MCP Server 爆发** | GitHub、Filesystem、PostgreSQL |

**关键词：** 定义、探索、各走各路

---

## 2025 年：爆发期

| 时间 | 事件 | 影响 |
|------|------|------|
| 2025 Q1 | **OpenAI Codex CLI 开源** | Apache 2.0，60K+ Stars |
| 2025 Q1 | **Roo Code 从 Cline fork** | 多 Mode + Cloud Agents |
| 2025 Q2 | **MCP 成为事实标准** | 3000+ 社区 MCP Server |
| 2025 Q2 | **Claude Code v2 发布** | Skills + Hooks + Subagents 系统 |
| 2025 Q3 | **AGENTS.md 开放标准发布** | Linux Foundation 背书，跨 Agent 兼容 |
| 2025 Q3 | **OpenHands SDK 化** | 多 Agent 编排 + MCP 支持 |
| 2025 Q4 | **Plugin 市场开始形成** | ClaudePluginHub 上线 |
| 2025 Q4 | **Kilo Code 获得 $8M 种子轮** | 500+ 模型 + 并行 Agent |

**关键词：** 竞争、标准化、生态形成

---

## 2026 年：成熟期

| 时间 | 事件 | 影响 |
|------|------|------|
| 2026 Q1 | **Claude Code v2.1.x** | 29 种 Hooks、Agent Teams、热加载 Skills |
| 2026 Q1 | **Codex CLI Context 达到 1M tokens** | 长上下文处理能力 |

**关键词：** 整合、共识、工具化

## 发展趋势

```
2024 ─────── 2025 ─────── 2026 ──────→
   │             │             │
   ▼             ▼             ▼
  定义期        竞争期        整合期
  ─────         ─────         ─────
  MCP 诞生      MCP 标准      MCP 生态 3000+
  Agent 原型    Agent 百花齐放 Agent 分层共识
  手动配置      Skills 出现    Skills 生态 1000+
  单 Agent      多 Agent      多 Agent 协调
```

### 未来方向

1. **Skills 标准化** — AGENTS.md 跨 Agent 兼容
2. **MCP 深化** — 安全层、审计层、缓存层
3. **Memory 统一** — 长期记忆成为 Agent 标配
4. **Plugin 跨平台** — 一个 Plugin 多平台运行
5. **Agent Marketplace** — 完整的 Agent 交易市场
